June 2024 web platform update

Now widely available on the web platform
Widely available

  • SharedArrayBuffer and Atomics: The SharedArrayBuffer object represents bytes shared between multiple workers and the main thread. The Atomics object safely accesses SharedArrayBuffer data to make sure predictable values are read and written and that operations are not interrupted. Learn more.

Newly available on the web platform
Newly available

  • Async clipboard: The navigator.clipboard API asynchronously reads and writes to the system clipboard. Learn more.

  • Set methods: The difference(), intersection(), isDisjointFrom(), isSubsetOf(), isSupersetOf(), symmetricDifference(), and union() methods of the JavaScript Set object performs operations between two sets. Learn more.
