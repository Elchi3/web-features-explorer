May 2024 web platform update

Newly available on the web platform
Newly available

  • light-dark(): The light-dark() CSS function accepts two colors and uses one depending on the current color scheme. Learn more.

  • round(), mod(), and rem(): The round(), mod(), and rem() CSS functions compute rounded values and the remainder after division. Learn more.

  • :state(): The :state() CSS pseudo-class matches custom elements based on their custom state, set through the ElementInternals.states API. Learn more.

  • text-wrap: balance: The text-wrap: balance CSS declaration balances the length of each line when text is broken into multiple lines. Also known as headline balancing. Learn more.

  • zoom: The zoom CSS property scales the size of an element. Unlike the transform property, a zoomed element affects page layout. Learn more.
