all

Widely available
Tags: css Groups: CSS > Explicit defaults

The all CSS property is a shorthand for all CSS properties, except for direction and unicode-bidi. It accepts only the keywords for explicit defaulting (such as initial and inherit), since they are the only values supported on all CSS properties. To learn more, see all at MDN.
