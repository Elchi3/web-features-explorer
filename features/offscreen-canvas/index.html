Offscreen canvas

Newly available since 2023-03-27
Tags: api Groups: Canvas

The OffscreenCanvas API provides a canvas that can be drawn to off screen, with no dependencies on the DOM, which can be used to run heavy rendering operations inside a worker context. To learn more, see HTMLCanvasElement: transferControlToOffscreen() method, OffscreenCanvas, OffscreenCanvas: OffscreenCanvas() constructor, OffscreenCanvas: convertToBlob() method, OffscreenCanvas: getContext() method, OffscreenCanvas: height property, OffscreenCanvas: transferToImageBitmap() method, OffscreenCanvas: width property, OffscreenCanvasRenderingContext2D at MDN.
