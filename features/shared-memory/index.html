SharedArrayBuffer and Atomics

Widely available
Tags: js Groups: JavaScript

The SharedArrayBuffer object represents bytes shared between multiple workers and the main thread. The Atomics object safely accesses SharedArrayBuffer data to make sure predictable values are read and written and that operations are not interrupted. To learn more, see Atomics, Atomics.add(), Atomics.and(), Atomics.compareExchange(), Atomics.exchange(), Atomics.isLockFree(), Atomics.load(), Atomics.notify(), Atomics.or(), Atomics.store(), Atomics.sub(), Atomics.wait(), Atomics.xor(), SharedArrayBuffer, SharedArrayBuffer[Symbol.species], SharedArrayBuffer() constructor, SharedArrayBuffer.prototype.byteLength, SharedArrayBuffer.prototype.slice() at MDN.
