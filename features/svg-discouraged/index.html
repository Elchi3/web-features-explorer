SVG 1.1 (discouraged)

Discouraged
Tags: api css svg

This feature is discouraged. Consider using SVG , @font-face instead. For the rationale, see:

  • https://svgwg.org/svg2-draft/changes.html#substantial

The SVG 1.1 image format has several components that were excluded from SVG 2, such as fonts, alternate glyphs, and the xlink namespace.
