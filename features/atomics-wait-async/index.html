Atomics.waitAsync

Limited availability
Tags: javascript Groups: JavaScript

The Atomics.waitAsync() static method waits for a value in a shared memory location, providing a promise when the expected value is not yet in memory. The waitAsync() method is a non-blocking alternative to Atomics.wait(). To learn more, see Atomics.waitAsync() at MDN.
