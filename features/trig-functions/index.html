sin(), cos(), tan(), asin(), acos(), atan(), and atan2() (CSS)

Newly available since 2023-03-13
Tags: css Groups: CSS

The sin(), cos(), tan(), asin(), acos(), atan(), and atan2() CSS functions compute various trigonometric functions. To learn more, see acos(), asin(), atan(), atan2(), cos(), sin(), tan() at MDN.
