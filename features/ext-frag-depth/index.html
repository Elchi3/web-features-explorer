EXT_frag_depth WebGL extension

Limited availability
Tags: api Groups: WebGL > WebGL extensions

The EXT_frag_depth extension for WebGL 1.0 and 2.0 contexts adds setting the depth value of a fragment from within the fragment shader. To learn more, see EXT_frag_depth extension at MDN.
