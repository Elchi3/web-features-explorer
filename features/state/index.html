:state()

Newly available since 2024-05-17
Tags: api css Groups: Custom elements HTML

The :state() CSS pseudo-class matches custom elements based on their custom state, set through the ElementInternals.states API. To learn more, see CustomStateSet, CustomStateSet: add() method, CustomStateSet: clear() method, CustomStateSet: delete() method, CustomStateSet: entries() method, CustomStateSet: forEach() method, CustomStateSet: has() method, CustomStateSet: keys() method, CustomStateSet: size property, CustomStateSet: values() method, ElementInternals: states property, :state().
