Multi-column layout

Widely available since 2019-09-07
Tags: css Groups: CSS > Layout > multi-column

Multi-column layout flows an element's content across one or more columns in a single row, without affecting the display property of its children.

To learn more, see column-count, column-gap, column-rule, column-rule-color, column-rule-style, column-rule-width, column-width, columns at MDN.
