video-dynamic-range media query

Limited availability
Tags: css Groups: CSS > Media queries

The video-dynamic-range CSS media query sets styles based on whether a device can display video with at least standard range colors or at least high range colors. Non-visual devices will match neither. To learn more, see video-dynamic-range at MDN.
