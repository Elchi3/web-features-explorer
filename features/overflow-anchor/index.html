overflow-anchor

Limited availability
Tags: css Groups: Scrolling

The overflow-anchor CSS property sets an element as a possible scroll anchor, reducing unintended scrolling when document changes occur above the current scrollport. This is enabled by default where supported. To learn more, see overflow-anchor at MDN.
