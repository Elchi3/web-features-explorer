willReadFrequently

Newly available since 2024-09-16
Tags: api Groups: Canvas

The optional willReadFrequently parameter of a canvas's getContext() method permits the browser to optimize for frequent getImageData() calls by avoiding hardware acceleration. Also known as multiple readback. To learn more, see HTMLCanvasElement: getContext() method at MDN.
