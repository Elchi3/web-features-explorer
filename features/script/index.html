<script> and <noscript>

Widely available
Tags: api html Groups: HTML > HTML elements

The <script> element contains or loads data or executable code. This is typically used to load JavaScript code. The <noscript> element represents alternative content to show when scripting is not allowed. To learn more, see HTMLScriptElement, HTMLScriptElement: async property, HTMLScriptElement: crossOrigin property, HTMLScriptElement: defer property, HTMLScriptElement: integrity property, HTMLScriptElement: src property, HTMLScriptElement: supports() static method, HTMLScriptElement: text property, HTMLScriptElement: type property, <noscript>: The Noscript element, <script>: The Script element, <script>: type attribute at MDN.
