Subgrid

Newly available since 2023-09-15
Tags: css Groups: Grid Layout CSS

The subgrid value for the grid-template-columns and grid-template-rows properties allows a grid item to inherit the grid definition of its parent grid container. To learn more, see Subgrid at MDN.
