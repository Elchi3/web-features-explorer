Weak references

Widely available
Tags: js Groups: JavaScript

The WeakRef and FinalizationRegistry objects hold references to garbage-collectable objects without creating strong references that prevent their garbage collection. To learn more, see FinalizationRegistry, FinalizationRegistry() constructor, FinalizationRegistry.prototype.register(), FinalizationRegistry.prototype.unregister(), WeakRef, WeakRef() constructor, WeakRef.prototype.deref().
