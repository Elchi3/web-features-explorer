Sticky positioning

Widely available
Tags: css Groups: CSS > Layout > Positioning

The position: sticky CSS declaration positions an element in the normal flow until it crosses a specified threshold, at which points it becomes fixed (stuck) at that position. To learn more, see position at MDN.
