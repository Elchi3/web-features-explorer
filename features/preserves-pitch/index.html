preservesPitch

Newly available since 2023-12-11
Tags: api Groups: HTML > Media elements

The preservesPitch property for <audio> or <video> adjusts the pitch of audio to sound more natural when the playback rate is faster or slower than the default. To learn more, see HTMLMediaElement: preservesPitch property at MDN.
