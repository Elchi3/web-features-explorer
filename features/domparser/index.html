DOMParser

Widely available since 2018-09-21
Tags: api Groups: Parsing and serialization

The DOMParser API parses XML or HTML source code from a string into a DOM Document.

To learn more, see DOMParser, DOMParser: DOMParser() constructor, DOMParser: parseFromString() method at MDN.
