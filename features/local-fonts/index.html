Local fonts

Limited availability
Tags: api html http

The window.queryLocalFonts() method returns an array of locally-installed fonts, each represented by a FontData object. To learn more, see FontData, FontData: blob() method, FontData: family property, FontData: fullName property, FontData: postscriptName property, FontData: style property, Local Font Access API, Permissions-Policy: local-fonts at MDN.
