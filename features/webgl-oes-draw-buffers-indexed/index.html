OES_draw_buffers_indexed WebGL extension

Newly available since 2022-12-13
Tags: api Groups: WebGL > WebGL extensions

The OES_draw_buffers_indexed extension for WebGL 2.0 contexts allows you to control blending on a per-color basis when writing to multiple color buffers simultaneously. To learn more, see OES_draw_buffers_indexed at MDN.
