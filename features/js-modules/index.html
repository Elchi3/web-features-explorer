JavaScript modules

Widely available
Tags: html js Groups: JavaScript modules

JavaScript modules allow code to be organized into reusable units. Modules use import to load other modules and export to declare what is available to import from other modules. In HTML, modules are loaded with <script type="module">. To learn more, see export, import at MDN.
