XMLHttpRequest

Widely available since 2018-01-29 (baseline high)

The XMLHttpRequest API makes HTTP requests. It's the predecessor to fetch(). Also known as XHR.

Compatibility

  • Chrome 1 Released on 2008-12-11
  • Chrome Android 18 Released on 2012-06-27

  • Firefox 1 Released on 2004-11-09
