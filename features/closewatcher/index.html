CloseWatcher

Limited availability
Tags: api

The CloseWatcher API listens and responds to close requests on in-page components, such as modals, popovers, and sidebars. To learn more, see CloseWatcher, CloseWatcher: CloseWatcher() constructor, CloseWatcher: cancel event, CloseWatcher: close() method, CloseWatcher: close event, CloseWatcher: destroy() method, CloseWatcher: requestClose() method at MDN.
