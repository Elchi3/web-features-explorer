Typed arrays (initial support)

Widely available
Tags: js Groups: Typed arrays Collections JavaScript

Typed arrays are ordered lists of JavaScript values, where all values are of the same numerical type, such as 8-bit integers or 32-bit floating point numbers. To learn more, see Float32Array, Float32Array() constructor, Float64Array, Float64Array() constructor, Int16Array, Int16Array() constructor, Int32Array, Int32Array() constructor, Int8Array, Int8Array() constructor, TypedArray, TypedArray.BYTES_PER_ELEMENT, TypedArray.prototype.buffer, TypedArray.prototype.byteLength, TypedArray.prototype.byteOffset, TypedArray.prototype.join(), TypedArray.prototype.length, TypedArray.prototype.reverse(), TypedArray.prototype.set(), TypedArray.prototype.slice(), TypedArray.prototype.sort(), TypedArray.prototype.subarray(), TypedArray.prototype.toString(), Uint16Array, Uint16Array() constructor, Uint32Array, Uint32Array() constructor, Uint8Array, Uint8Array() constructor, Uint8ClampedArray, Uint8ClampedArray() constructor at MDN.
