Fetch priority

Limited availability
Tags: api html http

The fetch() priority option and the fetchPriority HTML attribute give hints to the browser about which requests to do before other requests of the same type. To learn more, see HTMLImageElement: fetchPriority property, HTMLLinkElement: fetchPriority property, HTMLScriptElement: fetchPriority property at MDN.
