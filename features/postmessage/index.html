postMessage

Widely available
Tags: api Groups: Messaging

The postMessage() global method sends cross-origin messages to windows and workers, including popups and iframes. Also known as cross-document messaging. To learn more, see DedicatedWorkerGlobalScope: message event, DedicatedWorkerGlobalScope: postMessage() method, MessageEvent, MessageEvent: MessageEvent() constructor, MessageEvent: data property, MessageEvent: lastEventId property, MessageEvent: origin property, MessageEvent: source property, Window: message event, Window: postMessage() method, Worker: message event, Worker: postMessage() method at MDN.
