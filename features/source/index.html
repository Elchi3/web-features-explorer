<source>

Widely available since 2018-01-29
Tags: html Groups: HTML > Media elements HTML > HTML elements

The <source> element sets a media resource for the <picture>, <video>, and <audio> elements.

To learn more, see <source>: The Media or Image Source element at MDN.
