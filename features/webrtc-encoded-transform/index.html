WebRTC encoded transform

Limited availability
Tags: api Groups: WebRTC

The WebRTC encoded transform API allows you to modify audio and video streams in WebRTC connections. For example, it can be used for visual effects or custom codecs. To learn more, see DedicatedWorkerGlobalScope: rtctransform event, RTCEncodedAudioFrame, RTCEncodedAudioFrame: data property, RTCEncodedAudioFrame: getMetadata() method, RTCEncodedVideoFrame, RTCEncodedVideoFrame: data property, RTCEncodedVideoFrame: getMetadata() method, RTCEncodedVideoFrame: type property, RTCRtpReceiver: transform property, RTCRtpScriptTransform, RTCRtpScriptTransform: RTCRtpScriptTransform() constructor, RTCRtpScriptTransformer, RTCRtpScriptTransformer: generateKeyFrame(), RTCRtpScriptTransformer: options property, RTCRtpScriptTransformer: readable property, RTCRtpScriptTransformer: sendKeyFrameRequest(), RTCRtpScriptTransformer: writable property, RTCRtpSender: transform property, RTCTransformEvent, RTCTransformEvent: transformer property at MDN.
