hashchange

Widely available since 2018-01-29
Tags: api

The hashchange event fires when the URL fragment identifier (the part of the URL starting with #) of the current page has changed.

To learn more, see HashChangeEvent, HashChangeEvent: HashChangeEvent() constructor, HashChangeEvent: newURL property, HashChangeEvent: oldURL property, Window: hashchange event at MDN.
