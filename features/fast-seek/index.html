fastSeek()

Limited availability
Tags: api Groups: HTML > Media elements

The fastSeek() method seeks an <audio> or <video> element as fast as possible, by seeking to a keyframe instead of exactly the requested time. To learn more, see HTMLMediaElement: fastSeek() method at MDN.
