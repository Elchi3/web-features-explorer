localStorage and sessionStorage

Widely available
Tags: api Groups: Storage

The localStorage and sessionStorage APIs store data as key-value pairs. While localStorage persists across sessions, sessionStorage data is discarded when the page is closed. To learn more, see Storage, Storage: clear() method, Storage: getItem() method, Storage: key() method, Storage: length property, Storage: removeItem() method, Storage: setItem() method, StorageEvent, StorageEvent: key property, StorageEvent: newValue property, StorageEvent: oldValue property, StorageEvent: storageArea property, StorageEvent: url property, Window: localStorage property, Window: sessionStorage property, Window: storage event at MDN.
