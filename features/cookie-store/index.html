Cookie store

Limited availability
Tags: api

The CookieStore interface is an asynchronous and promise-based API for managing cookies. It does not rely on document and so is available to service workers as well.

To learn more, see CookieChangeEvent, CookieChangeEvent: CookieChangeEvent() constructor, CookieChangeEvent: changed property, CookieChangeEvent: deleted property, CookieStore, CookieStore: change event, CookieStore: delete() method, CookieStore: get() method, CookieStore: getAll() method, CookieStore: set() method, CookieStoreManager, CookieStoreManager: getSubscriptions() method, CookieStoreManager: subscribe() method, CookieStoreManager: unsubscribe() method, ExtendableCookieChangeEvent, ExtendableCookieChangeEvent: ExtendableCookieChangeEvent() constructor, ExtendableCookieChangeEvent: changed property, ExtendableCookieChangeEvent: deleted property, ServiceWorkerGlobalScope: cookieStore property, ServiceWorkerGlobalScope: cookiechange event, ServiceWorkerRegistration: cookies property, Window: cookieStore property at MDN.
