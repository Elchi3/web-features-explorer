display-mode media query

Limited availability
Tags: css Groups: Media queries CSS

The display-mode CSS media query sets styles based on whether the web page is in fullscreen, standalone, minimal-ui, browser, or picture-in-picture mode. To learn more, see display-mode at MDN.
