Functions

Widely available
Tags: js Groups: JavaScript

Functions are series of statements that can be called and return a value. The function keyword (as in function () { }) and arrow (=>) expression create functions. The JavaScript functions protocol includes default and rest parameters and binding to this. To learn more, see Function, Function.prototype[Symbol.hasInstance](), Function() constructor, Function.prototype.apply(), Function.prototype.bind(), Function.prototype.call(), Function: length, Function: name, Function.prototype.toString(), Functions, The arguments object, arguments[Symbol.iterator](), arguments.length, Arrow function expressions, Default parameters, get, Method definitions, Rest parameters, set, function expression, function.
