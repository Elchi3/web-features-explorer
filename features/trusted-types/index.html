Trusted types

Limited availability
Tags: api http

Trusted types allow you to lock down insecure parts of the DOM API and prevent client-side cross-site scripting (XSS) attacks. To learn more, see TrustedHTML, TrustedHTML: toJSON() method, TrustedHTML: toString() method, TrustedScript, TrustedScript: toJSON() method, TrustedScript: toString() method, TrustedScriptURL, TrustedScriptURL: toJSON() method, TrustedScriptURL: toString() method, TrustedTypePolicy, TrustedTypePolicy: createHTML() method, TrustedTypePolicy: createScript() method, TrustedTypePolicy: createScriptURL() method, TrustedTypePolicy: name property, TrustedTypePolicyFactory, TrustedTypePolicyFactory: createPolicy() method, TrustedTypePolicyFactory: defaultPolicy property, TrustedTypePolicyFactory: emptyHTML property, TrustedTypePolicyFactory: emptyScript property, TrustedTypePolicyFactory: getAttributeType() method, TrustedTypePolicyFactory: getPropertyType() method, TrustedTypePolicyFactory: isHTML() method, TrustedTypePolicyFactory: isScript() method, TrustedTypePolicyFactory: isScriptURL() method, Trusted Types API, CSP: require-trusted-types-for, CSP: trusted-types.
