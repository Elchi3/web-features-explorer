requestVideoFrameCallback()

Limited availability
Tags: api Groups: HTML > Media elements

The requestVideoFrameCallback() method for <video> schedules a function that runs with the next video frame. It is similar to requestAnimationFrame(), but for video. To learn more, see HTMLVideoElement: cancelVideoFrameCallback() method, HTMLVideoElement: requestVideoFrameCallback() method at MDN.
