Conic gradients

Widely available
Groups: CSS > Gradients

The conic-gradient() and repeating-conic-gradient() CSS functions create backgrounds that progress between two or more colors around a center point. To learn more, see conic-gradient(), repeating-conic-gradient() at MDN.
