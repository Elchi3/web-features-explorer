scrollbar-gutter

Limited availability
Tags: css Groups: Scrolling

The scrollbar-gutter CSS property reserves space for the scrollbar, preventing unwanted layout changes as the scrollbar appears and disappears. To learn more, see scrollbar-gutter at MDN.
