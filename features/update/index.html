Update frequency media query

Newly available since 2023-09-18
Tags: css Groups: CSS > Media queries

The update CSS media query sets styles based on whether and how fast the user's device can modify display after it has been rendered. For example, you can avoid animations on devices that aren't fast enough to display them smoothly. To learn more, see update at MDN.
