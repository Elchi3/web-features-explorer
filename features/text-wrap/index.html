text-wrap

Newly available since 2024-03-19
Tags: css Groups: CSS > Text > Text wrap

The text-wrap CSS property sets how lines break in text that overflows the container. It is a shorthand for text-wrap-style and text-wrap-mode. To learn more, see text-wrap at MDN.
