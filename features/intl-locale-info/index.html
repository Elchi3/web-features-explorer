Intl.Locale info

Limited availability
Tags: js Groups: Internationalization JavaScript

The getter methods of the Intl.Locale API provide supplemental information about a Unicode locale, such as the locale's calendar (for example, the first day in a week or the weekend start day), writing direction, 12- or 24-hour cycles, and numbering system. To learn more, see Intl.Locale at MDN.
