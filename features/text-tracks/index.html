Text tracks

Widely available
Tags: api html Groups: HTML > Media elements HTML > HTML elements

The <track> element is used as a child of the media elements that lets you specify a timed text track to be displayed in parallel with the media element. To learn more, see <track>: The Embed Text Track element at MDN.
