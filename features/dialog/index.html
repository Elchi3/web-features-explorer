<dialog>

Widely available
Tags: api html Groups: HTML > HTML elements

The <dialog> HTML element represents a modal or non-modal dialog box, such as a confirmation prompt or a subwindow used to enter data. To learn more, see <dialog>: The Dialog element at MDN.
