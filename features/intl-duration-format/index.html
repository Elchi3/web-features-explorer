Intl.DurationFormat

Limited availability
Tags: js Groups: Internationalization JavaScript

The Intl.DurationFormat API creates a locale-aware formatter that turns an object representing a duration (such as days, hours, and minutes) into a string. To learn more, see Intl.DurationFormat at MDN.
