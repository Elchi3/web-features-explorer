Origin private file system

Newly available since 2023-03-14
Tags: api Groups: File system access

The navigator.storage.getDirectory() method returns a FileSystemDirectoryHandle that is restricted to a specific origin and invisible to the user's actual file system for faster file-based applications, such as SQLite databases. To learn more, see FileSystemDirectoryHandle, FileSystemDirectoryHandle: entries() method, FileSystemDirectoryHandle: getDirectoryHandle() method, FileSystemDirectoryHandle: getFileHandle() method, FileSystemDirectoryHandle: keys() method, FileSystemDirectoryHandle: removeEntry() method, FileSystemDirectoryHandle: resolve() method, FileSystemDirectoryHandle: values() method, FileSystemFileHandle, FileSystemFileHandle: createSyncAccessHandle() method, FileSystemFileHandle: getFile() method, FileSystemHandle, FileSystemHandle: isSameEntry() method, FileSystemHandle: kind property, FileSystemHandle: name property, FileSystemSyncAccessHandle, FileSystemSyncAccessHandle: close() method, FileSystemSyncAccessHandle: flush() method, FileSystemSyncAccessHandle: getSize() method, FileSystemSyncAccessHandle: read() method, FileSystemSyncAccessHandle: truncate() method, FileSystemSyncAccessHandle: write() method, StorageManager: getDirectory() method at MDN.
