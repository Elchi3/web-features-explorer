User agent client hints

Limited availability
Tags: api http

The Sec-CH-UA HTTP request header and the navigator.userAgentData API expose browser and platform information used to vary responses and application logic. User agent client hints are similar to user agent strings but more privacy-preserving and safer to parse. To learn more, see User-Agent Client Hints API at MDN.
