Dataset

Widely available
Tags: api

The dataset API attaches and accesses custom data on elements, using either the dataset property of HTMLElement or attributes prefixed with data-. To learn more, see DOMStringMap, HTMLElement: dataset property, SVGElement: dataset property at MDN.
