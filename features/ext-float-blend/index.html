EXT_float_blend WebGL extension

Limited availability
Tags: api Groups: WebGL > WebGL extensions

The EXT_float_blend extension for WebGL 1.0 and 2.0 contexts adds 32-bit floating-point components to blending and draw buffers. To learn more, see EXT_float_blend extension at MDN.
