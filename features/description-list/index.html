Description list

Widely available
Tags: api html Groups: HTML

The <dl>, <dt>, and '' HTML elements represent a list of terms and their associated descriptions. Use a description list to create glossaries or similar list of key-value pairs.

To learn more, see HTMLDListElement, <dd>: The Description Details element, <dl>: The Description List element, <dt>: The Description Term element at MDN.
