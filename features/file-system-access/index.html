File system access

Limited availability
Tags: api Groups: File system access

The showOpenFilePicker(), showDirectoryPicker(), and showSaveFilePicker() methods request access to files and directories on the user's device and returns a handle for reading and writing to them. To learn more, see File System API at MDN.
