Resource timing (initial support)

Widely available
Tags: api Groups: Resource timing

PerformanceResourceTiming entries report when network events happen while loading a resource, such as when connections start and end. You can use this information to measure loading times. To learn more, see PerformanceResourceTiming, PerformanceResourceTiming: connectEnd property, PerformanceResourceTiming: connectStart property, PerformanceResourceTiming: domainLookupEnd property, PerformanceResourceTiming: domainLookupStart property, PerformanceResourceTiming: fetchStart property, PerformanceResourceTiming: initiatorType property, PerformanceResourceTiming: redirectEnd property, PerformanceResourceTiming: redirectStart property, PerformanceResourceTiming: requestStart property, PerformanceResourceTiming: responseEnd property, PerformanceResourceTiming: responseStart property at MDN.
