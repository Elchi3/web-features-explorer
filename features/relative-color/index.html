Relative colors

Newly available since 2024-07-09
Tags: css Groups: CSS

The from keyword for color functions (color(), hsl(), oklch(), etc.) creates a new color based on a given color by modifying the values of the input color. Also known as relative color syntax (RCS). To learn more, see Using relative colors at MDN.
