Constructed stylesheets

Newly available since 2023-03-27
Tags: api

The CSSStyleSheet constructor creates a new stylesheet which can be shared between a document and multiple shadow roots using adoptedStyleSheets. To learn more, see Document: adoptedStyleSheets property at MDN.
