SMIL SVG animations

Widely available
Tags: api svg Groups: Animation SVG

The <animate>, <animateMotion>, and <animateTransform> SVG elements declaratively animate SVG elements. Also known as SMIL. To learn more, see SVG animation with SMIL at MDN.
