SMIL SVG animations

Widely available
Tags: api svg Groups: Animation SVG

The <animate>, <animateMotion>, and <animateTransform> SVG elements declaratively animate SVG elements. Also known as SMIL. To learn more, see SVGAnimateColorElement, SVGAnimateElement, SVGAnimateMotionElement, SVGAnimateTransformElement, SVGAnimatedAngle, SVGAnimatedBoolean, SVGAnimatedEnumeration, SVGAnimatedEnumeration: animVal property, SVGAnimatedEnumeration: baseVal property, SVGAnimatedInteger, SVGAnimatedLength, SVGAnimatedLength: animVal property, SVGAnimatedLength: baseVal property, SVGAnimatedLengthList, SVGAnimatedNumber, SVGAnimatedNumberList, SVGAnimatedPreserveAspectRatio, SVGAnimatedRect, SVGAnimatedString, SVGAnimatedString: animVal property, SVGAnimatedString: baseVal property, SVGAnimatedTransformList, SVGAnimationElement, SVGAnimationElement: beginEvent event, SVGAnimationElement: endEvent event, SVGAnimationElement: repeatEvent event, SVGAnimationElement: targetElement property, <animate>, dur, from, repeatCount, <animateMotion>, keyPoints, <animateTransform>, by at MDN.
