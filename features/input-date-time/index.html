Date and time <input> types

Widely available
Tags: api html Groups: HTML > Forms HTML > HTML elements

The <input type="date"> and <input type="time"> HTML elements show date and time pickers. To learn more, see HTMLInputElement: valueAsDate property, <input type="date">, <input type="datetime-local">, <input type="month">, <input type="time">, <input type="week"> at MDN.
