WebHID

Limited availability
Tags: api http

The WebHID API provides access to Human Interface Devices (HID) that are connected to the user's device. To learn more, see HID, HID: connect event, HID: disconnect event, HID: getDevices() method, HID: requestDevice() method, HIDConnectionEvent, HIDConnectionEvent: HIDConnectionEvent() constructor, HIDConnectionEvent: device property, HIDDevice, HIDDevice: close() method, HIDDevice: collections property, HIDDevice: inputreport event, HIDDevice: open() method, HIDDevice: opened property, HIDDevice: productId property, HIDDevice: productName property, HIDDevice: receiveFeatureReport() method, HIDDevice: sendFeatureReport() method, HIDDevice: sendReport() method, HIDDevice: vendorId property, HIDInputReportEvent, HIDInputReportEvent: data property, HIDInputReportEvent: device property, HIDInputReportEvent: reportId property, Navigator: hid property, Permissions-Policy: hid.
