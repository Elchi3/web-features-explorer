WeakMap

Widely available
Tags: javascript Groups: JavaScript > Collections

A WeakMap object holds key-value pairs that do not create strong references to its keys, such that value data can be associated with a key without preventing garbage collection of the key. To learn more, see WeakMap at MDN.
