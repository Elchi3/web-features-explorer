prefers-reduced-transparency media query

Limited availability
Tags: css http Groups: CSS > Media queries

The prefers-reduced-transparency CSS media query sets styles based on whether the user prefers to reduce the amount of transparent effects on their device, for example to improve contrast and legibility. To learn more, see prefers-reduced-transparency, Sec-CH-Prefers-Reduced-Transparency at MDN.
