Font metric overrides

Limited availability
Tags: api css Groups: CSS > Fonts

The ascent-override, descent-override, and line-gap-override descriptors adjust the display of a font declared in an @font-face statement. This can help make fallback fonts look the same size. To learn more, see FontFace: ascentOverride property, FontFace: descentOverride property, FontFace: lineGapOverride property, ascent-override, descent-override, line-gap-override at MDN.
