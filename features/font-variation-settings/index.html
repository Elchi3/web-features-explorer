font-variation-settings

Widely available
Tags: css Groups: CSS > Fonts

The font-variation-settings CSS property sets an "axis of variability" on a variable font, such as weight, optical size, or a custom axis defined by the typeface designer. When possible, use other CSS font properties, such as font-weight: bold. Also known as variable fonts. To learn more, see font-variation-settings at MDN.
