JSON source text access

Limited availability
Tags: javascript Groups: JavaScript > JSON

To serialize and parse JSON in a lossless way, JSON.stringify() handles rawJSON values and JSON.parse()'s reviver callback takes a source context parameter. To learn more, see JSON.isRawJSON(), JSON.rawJSON() at MDN.
