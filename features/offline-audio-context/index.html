OfflineAudioContext

Widely available
Tags: api Groups: Web Audio

The OfflineAudioContext API represents an audio processing graph that writes to an AudioBuffer instead of an output device. To learn more, see OfflineAudioCompletionEvent, OfflineAudioCompletionEvent: OfflineAudioCompletionEvent() constructor, OfflineAudioCompletionEvent: renderedBuffer property, OfflineAudioContext, OfflineAudioContext: OfflineAudioContext() constructor, OfflineAudioContext: complete event, OfflineAudioContext: length property, OfflineAudioContext: resume() method, OfflineAudioContext: startRendering() method, OfflineAudioContext: suspend() method at MDN.
