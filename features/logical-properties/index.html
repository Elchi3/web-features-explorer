Logical properties

Widely available
Tags: css Groups: CSS

CSS logical properties control borders, size, margin, and padding with directions and dimensions relative to the writing mode. For example, in a left to right, top to bottom writing mode, block-end refers to the bottom. Also known as flow relative. To learn more, see CSS logical properties and values at MDN.
