Page selectors

Limited availability
Tags: css Groups: CSS > Selectors Print media

The :first, :left, and :right pseudo-classes select pages based on their position in sequence after pagination. They're often used with the page CSS property, to choose a print layout defined by the @page rule. To learn more, see page, :first, :left, :right at MDN.
