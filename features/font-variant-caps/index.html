font-variant-caps

Widely available
Tags: css Groups: CSS > Fonts > Typographic font features

The font-variant-caps CSS property sets whether text should be displayed in small caps, petite caps, or with capital letters designed for titles. To learn more, see font-variant-caps at MDN.
