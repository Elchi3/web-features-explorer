MathML

Newly available since 2023-01-12
Tags: api css mathml

MathML, or the Mathematical Markup Language, describes mathematical notation, such as expressions and formulas. Also known as MathML Core. To learn more, see MathMLElement, MathMLElement: attributeStyleMap property, MathMLElement: style property, math-depth, math-shift, math-style, <annotation>, <annotation-xml>, MathML, <menclose>, <merror>, <mfrac>, <mi>, <mmultiscripts>, <mn>, <mo>, <mover>, <mpadded>, <mphantom>, <mprescripts>, <mroot>, <mrow>, <ms>, <mspace>, <msqrt>, <mstyle>, <msub>, <msubsup>, <msup>, <mtable>, <mtd>, <mtext>, <mtr>, <munder>, <munderover>, <semantics>, dir, displaystyle, href, scriptlevel at MDN.
