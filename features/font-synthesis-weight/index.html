font-synthesis-weight

Newly available since 2023-03-27
Tags: css Groups: CSS > Fonts > Font synthesis

The font-synthesis-weight CSS property sets whether or not the browser should synthesize bold typefaces when they're missing from the font. To learn more, see font-synthesis-weight at MDN.
