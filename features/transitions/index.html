Transitions (CSS)

Widely available
Tags: api css Groups: CSS > Transitions (CSS)

The transition shorthand CSS property sets how changes to an element's styles may occur over time. Transitions can be applied to specific CSS properties, all properties, or none. To learn more, see CSSTransition, CSSTransition: transitionProperty property, Element: transitioncancel event, Element: transitionend event, Element: transitionrun event, Element: transitionstart event, TransitionEvent, TransitionEvent: TransitionEvent() constructor, TransitionEvent: elapsedTime property, TransitionEvent: propertyName property, TransitionEvent: pseudoElement property, transition, transition-delay, transition-duration, transition-property, transition-timing-function at MDN.
