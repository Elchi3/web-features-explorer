Keyboard map

Limited availability
Tags: api

The navigator.keyboard.getLayoutMap() API returns a map from key codes to human readable key names. It can be used to describe buttons to the user when using the keyboard as a set of buttons, such as in games. To learn more, see Keyboard: getLayoutMap() method, KeyboardLayoutMap.
