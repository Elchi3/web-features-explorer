Channel messaging

Widely available
Tags: api Groups: Messaging

Channel messaging communicates one-to-one between browsing contexts on the same origin, such as between <iframe> elements or two documents using a shared worker. To learn more, see MessageChannel, MessageChannel: MessageChannel() constructor, MessageChannel: port1 property, MessageChannel: port2 property, MessageEvent: ports property, MessagePort, MessagePort: close() method, MessagePort: message event, MessagePort: postMessage() method, MessagePort: start() method at MDN.
