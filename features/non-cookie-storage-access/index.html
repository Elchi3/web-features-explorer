Non-cookie storage access

Limited availability
Tags: api Groups: Storage

Non-cookie storage access extends the requestStorageAccess() API to let content in cross-site iframes request access to first-party data beyond cookies. To learn more, see Document: hasUnpartitionedCookieAccess() method, StorageAccessHandle, StorageAccessHandle: BroadcastChannel() property, StorageAccessHandle: SharedWorker() method, StorageAccessHandle: caches property, StorageAccessHandle: createObjectURL() property, StorageAccessHandle: estimate() property, StorageAccessHandle: getDirectory() property, StorageAccessHandle: indexedDB property, StorageAccessHandle: localStorage property, StorageAccessHandle: locks property, StorageAccessHandle: revokeObjectURL() property, StorageAccessHandle: sessionStorage property at MDN.
