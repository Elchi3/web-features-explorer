Shadow DOM

Widely available
Tags: api Groups: Web Components

Shadow DOM allows you to attach encapsulated "shadow" DOM trees to elements. A shadow DOM tree is a separate component, isolated from the scripts and styles in other parts of the document. This is a part of Web Components. To learn more, see Element: attachShadow() method, Element: shadowRoot property, Event: composed property, Event: composedPath() method, Node: getRootNode() method, Node: isConnected property, ShadowRoot, ShadowRoot: clonable property, ShadowRoot: host property, ShadowRoot: mode property at MDN.
