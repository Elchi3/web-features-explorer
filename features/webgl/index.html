WebGL

Widely available
Tags: api Groups: WebGL

The WebGLRenderingContext API is the "webgl" rendering context for the <canvas> element. It represents a space for drawing two- and three-dimensional graphics and animations. To learn more, see WebGL: 2D and 3D graphics for the web at MDN.
