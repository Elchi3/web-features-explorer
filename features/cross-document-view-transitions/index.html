Cross-document view transitions

Limited availability
Tags: api css Groups: View Transitions

The @view-transition CSS rule sets whether a document opts-in to transitions between documents in a multi-page application. Both the old and new document must opt-in for a transition to be triggered. To learn more, see @view-transition at MDN.
