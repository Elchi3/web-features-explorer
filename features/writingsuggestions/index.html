writingsuggestions

Limited availability
Tags: api html Groups: Forms HTML

The writingsuggestions HTML attribute turns on or off a browser's writing suggestions. Writing suggestions vary by browser. For example, if turned on then a browser might show inline text completions accepted by pressing Tab. To learn more, see writingsuggestions at MDN.
