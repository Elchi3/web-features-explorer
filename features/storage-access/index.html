Storage access

Newly available since 2023-12-05
Tags: api

The document.requestStorageAccess() method allows content in iframes to request storing and reading cookies and other site data, while the document.hasStorageAccess() method checks if such access is granted. To learn more, see Document: hasStorageAccess() method, Document: requestStorageAccess() method at MDN.
