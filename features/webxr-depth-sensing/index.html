Depth sensing

Limited availability
Tags: api Groups: WebXR

An XRDepthInformation object represents a measurement of the distance from the user's device to the real-world geometry in the user's environment. To learn more, see XRCPUDepthInformation, XRCPUDepthInformation: data property, XRCPUDepthInformation: getDepthInMeters() method, XRDepthInformation, XRDepthInformation: height property, XRDepthInformation: normDepthBufferFromNormView property, XRDepthInformation: rawValueToMeters property, XRDepthInformation: width property, XRFrame: getDepthInformation() method, XRSession: depthDataFormat property, XRSession: depthUsage property, XRWebGLBinding: getDepthInformation() method, XRWebGLDepthInformation, XRWebGLDepthInformation: texture property at MDN.
