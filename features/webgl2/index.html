WebGL2

Widely available
Tags: api

The WebGL2RenderingContext API is the "webgl2" rendering context for the <canvas> element. It represents a space for drawing two- and three-dimensional graphics and animations. It corresponds to OpenGL ES 3.0. To learn more, see WebGL: 2D and 3D graphics for the web at MDN.
